1. Navigate to XPay dashboard through the follwing Urls:

- [Testing dashboard](https://staging.xpay.app/admin/)

- [Testing dashboard](https://community.xpay.app/admin/)

<img alt="XPay dashboard login screen" src={useBaseUrl("guides/login.png")} />

2. Enter your credentials and click on **Log in**.

3. Click on **add API key** button in the top right corner.

4. Fill in the name.

<img alt="Add API key" src={useBaseUrl("guides/add_api_key.png")} />

5. click **Save**, you will be redirected back to **API keys**.

6. copy the api key value from the message that appears in the top and save it safely as it can't be retrieved later.

<img alt="API key" src={useBaseUrl("guides/new_api_key.png")} />
